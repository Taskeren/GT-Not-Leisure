## âœ¨ Key Mechanics

Based on the internal systems, GTNL introduces features that change how you build your base:

* **Parallel Control Cores**: Install specialized cores (LV to UMV) into machine controllers to drastically increase parallel processing and reduce power costs.
* **Wireless Mode**: High-tier multiblocks (like the *Grand Assembly Line*) can operate without Energy Hatches when equipped with UMV+ cores.
* **Perfect Overclock**: Certain endgame machines (e.g., *Mega Blast Furnace*) enable 100% efficient overclocking when specific temperature or tier thresholds are met.
* **Structural Scaling**: Machines like the *Space Elevator* and *Meteor Miner* gain new abilities and range based on the tier of casings used.

## âš ï¸ Installation & Update Instructions

> **IMPORTANT:** When updating `GT-Not-Leisure`, you **must** reset the configuration to prevent recipe conflicts.

1.  **Backup** your save/instance.
2.  Delete the old `.jar` from your mods folder.
3.  **Delete/Reset** the GT language file: `GregTech.lang` (The mod injects new entries here).
4.  **Delete/Reset** the config file: `config/GTNotLeisure/main.cfg`.
